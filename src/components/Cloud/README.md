# Cloud 컴포넌트 명세서 (Cloud Component Specifications)

## 1. 역할
Cloud 컴포넌트는 게임 화면 상공(하늘)에서 서로 다른 높이와 속도로 유유히 흘러가는 배경 데코레이션 구름들을 렌더링하는 비대화형 그래픽 요소입니다.

---

## 2. Props 인터페이스

```typescript
interface CloudProps {
  currentFrame: number;    // 누적 게임 루프 프레임 수
  scale: number;           // 디바이스 비율 스케일 인자
  containerWidth: number;  // 실제 모바일 화면의 가로 넓이
}
```

---

## 3. 세부 디자인 및 스크롤 연출
- **개별 흐름 제어:**
  - 3개의 서로 다른 구름 속성(시작 X좌표, Y 높이, 크기, 속도 비율)을 정의하여 단조로움을 피하고 입체감 있는 하늘 배경을 묘사합니다.
  - 각 구름은 프레임 틱에 따라 `left`가 서서히 마이너스로 가며, 화면을 완전히 벗어나면 오른쪽 끝(`VIRTUAL_WIDTH + 60`)으로 리셋되어 무한히 스크롤됩니다.
- **픽셀 아트 형상:**
  - 3개의 겹친 직사각형 바(`<View>`)를 통해 구름 특유의 몽실몽실한 픽셀 실루엣을 표현합니다.
  - 가독성을 저해하지 않고 메인 장애물과 시각적으로 혼동되지 않도록 연회색 바탕(`backgroundColor: '#9e9e9e'`)과 반투명도(`opacity: 0.35`)를 부여하여 배경과 조화롭게 블렌딩시킵니다.

---

## 4. 의존성 모듈 목록

- **React Native**: 절대 좌표 배치용 `View`, `StyleSheet`
